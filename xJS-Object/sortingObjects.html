Examples of how to sort Object

Below we will generate the Object, which will use for examples

Source code:

let myObj = {
obj1: {value: 3},
obj2: {value: 1},
obj3: {value: 2},
obj4: {value: 5},
obj5: {value: 4},
}

Code output:

{ obj1: { value: 3 }, obj2: { value: 1 }, obj3: { value: 2 }, obj4: { value: 5 }, obj5: { value: 4 } }

Sorted keys in the Object

Here we get only the keys sorted alphabetically

Source code:

let sortedKeys = Object.keys(myObj).sort()

Code output:

[ 'obj1', 'obj2', 'obj3', 'obj4', 'obj5' ]

Sorted keys in the Object by their nested values

We will sort ascending the keys of object based on the number value in their nested keys

Source code:

let sortedKeys = Object.keys(myObj).sort((a, b) => { let diff = myObj[a]["value"] > myObj[b]["value"] if (myObj[a]["value"] === myObj[b]["value"]) { return 0 } return diff }) return sortedKeys

Code output:

[ 'obj2', 'obj3', 'obj1', 'obj5', 'obj4' ]
